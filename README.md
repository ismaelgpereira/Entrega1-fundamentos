## Funcionalidades

- Estrutura **HTML5 semântica** com uso de `<header>`, `<main>`, `<section>` e `<footer>`.
- **Menu de navegação** entre páginas.
- **Formulário com validação HTML5** (`required`, `type`, `maxlength`, `pattern`).
- **Máscaras automáticas** de CPF, telefone e CEP via JavaScript.
- Layout responsivo e estilizado com **CSS puro**.
